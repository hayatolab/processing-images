### Pré-Requisitos

- python3

```bash
sudo apt install python3
```

- pip3

```bash
sudo apt install python3-pip
```

- matplotlib

```bash
pip3 install -U matplotlib
```

- opencv

```bash
pip3 install -U opencv-python
```

- numpy

```bash
pip3 install -U numpy
```

### Execução

Compilação do programa

```bash
python3 main.py
```
